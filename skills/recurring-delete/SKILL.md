# Delete Recurring Task

Deletes recurring task templates using `todu template delete`.

## Examples

### Delete by ID

**User**: "Delete recurring task #14"

1. Runs `todu template show 14` to get details
2. Shows: "Weekly review (active, Weekly on Fri)"
3. Asks: "Yes, delete permanently" / "Cancel"
4. Runs `todu template delete 14 --force`

### No ID provided

**User**: "Delete a recurring task"

1. Runs `todu template list --type task`
2. Asks which to delete
3. Confirms and deletes

## CLI Commands

```bash
# Show template details
todu template show <id>

# List recurring tasks
todu template list --type task

# Delete (--force skips CLI confirmation)
todu template delete <id> --force
```

## Confirmation Options

"Yes, delete permanently" / "Cancel"

If active, warn: "Currently active - will stop generating tasks"

## Notes

- Deletion is permanent
- Warn if template is active
- Does not delete tasks already generated by the template
